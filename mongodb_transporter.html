Summary
MongoDB Transporter is a Javascript application running on Node.js with a Web GUI used for exporting, importing, and backuping MongoDB databases.
Web GUI (Dashboard)
Main dashboard of the app. It enables you to connect to your MongoDB instance, and to select which database you want to export.
Web GUI (Export popup)
Here you can select which collections you want to include in the database export. Databases are exported into a .json file.
Web GUI (Import popup)
Here you can import databases previously exported with this tool. You can select whether you want to append documents to existing collections or overwrite collections with the importing file.
