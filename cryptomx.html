This was my first slightly larger project made with Java and JavaFX. I started working on it in 2019 after taking an interest in desktop application. The app retrieves crypto price data from the CoinMarketCap API and than accoring to that data it calculates the total value of your crypto portfolio. It features a chart that displays how your portfolio changed in value in the past.
